# Study Schedule Application

This project is a study schedule application designed for 12th standard students. It helps students manage their study sessions and subjects effectively.

## Features

- Create and manage study schedules
- List and manage subjects
- User-friendly interface

## Technologies Used

- Python
- Flask
- HTML/CSS
- JavaScript

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd study-schedule-app
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

4. Run the application:
   ```
   python src/app.py
   ```

5. Open your browser and go to `http://127.0.0.1:5000` to access the application.
